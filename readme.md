#SSHPANEL
###An easy panel to manage SSH User's.

This script is used to SSH seller to easily manage SSH user or create reseller.

## System Requirements

* PHP 5.2 >=
* PDO PHP Extension
* OpenSSL PHP Extension
* Mbstring PHP Exstension
* Tokenizer PHP Exstension
* XML PHP Exstension
* exec()
